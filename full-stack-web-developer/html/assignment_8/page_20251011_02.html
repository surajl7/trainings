HTML enctype Attributes - Default: application/x-www-form-urlencoded
application/x-www-form-urlencoded:
It is default encoding type if the enctype attribute is not included in the form. All characters are encoded before submitting the form.







enctype - multipart/form-data
multipart/form-data:
It does not encode any character. It is used when our form contains file-upload controls.







enctype - text/plain
In this encoding type only space are encoding into + symbol and no any other special character encoded.

HTML disabled attribute
The HTML disabled attribute when applied then it disable that input field. The disabled field not allow the user to interact with that field. The disabled input filed does not receive click events, and these input value will not be sent to the server when submitting the form.
