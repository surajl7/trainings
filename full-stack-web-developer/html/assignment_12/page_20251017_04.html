< meta name="description" content="Free Web Tutorials" >
If you give a meta description then it will be useful for the relevant search to perform by search engines.

To define keywords for search engines

< meta name="keywords" content="HTML, CSS, XML, JavaScript" >
The keywords value is also used to provide keywords for a search engine, but it may ignore by browser due to spammers.

To define auther of the webpage:

< meta name="auther" content="Akon" >
The auther value specifies the name of the person who write the page content, and it is useful to automatically extract auther information by some content managment systems.
